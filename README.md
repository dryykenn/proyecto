# Sorpresa para Ereein

Abre `index.html` con doble clic: se verá en Chrome, Edge, Safari o Firefox, sin usar una terminal.

## Música

El audio compartido ya está incluido como `audio.mpeg`. La página empezará a reproducirlo después del toque inicial de bienvenida; los navegadores bloquean el sonido automático sin una interacción de la persona.

## Enlace privado

Para obtener un enlace que puedas mandar, sube **el contenido de esta carpeta** a un alojamiento que permita acceso restringido (por ejemplo, un sitio protegido con contraseña o una carpeta privada compartida solo con la persona elegida). El enlace no se puede crear desde aquí porque debe quedar asociado a tu propia cuenta del servicio que elijas.

No publiques la canción ni el archivo `audio.mpeg` en un sitio público salvo que tengas los derechos necesarios.
